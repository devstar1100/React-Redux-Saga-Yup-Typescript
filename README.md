# Space Saas

React front-end app for Space Saas.

## Prerequisites

- [Node.js 16.20.0](https://nodejs.org/en/download/releases)
- [Yarn >=1.22.11](https://yarnpkg.com/getting-started/install#install-corepack)

## Installation

Use the package manager [yarn](https://yarnpkg.com/getting-started/install) to install all the packages.

```bash
yarn
```
or
```bash
yarn install
```

## Launch the dev mode of the app

```bash
yarn dev
```

## Build the app to use it in production mode

```bash
yarn build
```

## Additional commands which are useful for development

### Run linter through the code of the entire app

```bash
yarn lint
```

### Check the validity of types used in the code of the entire app

```bash
yarn type-check
```

### Format the code of the entire app

```bash
yarn prettier
```
